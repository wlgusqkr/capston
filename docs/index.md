## 핵심 기능

- 행정동 히트맵 (425개 동, 4단계 색)
- 가중치 슬라이더 (전월세/생활시설/교통, 합 100% 자동 보정)
- 5쌍 비교로 가중치 자동 추정 (Bradley-Terry + scipy SLSQP)
- 동네 패널 + 6 섹션 상세
- 줌 13+ 실거래 핀
- 동네 비교 (최대 3개)
- 마이페이지 (Django session 인증, 찜/가중치 저장)
- 다크/라이트 모드
